## 🚀 Quick Start (3 Commands)

```bash
# 1. Install & start backend
npm install && npm start

# 2. Install & start frontend (new terminal)
cd frontend && npm install && npm start

# 3. Open browser
# Visit: http://localhost:3000
```

**That's it!** Your simulation is running locally.

## 🎯 Core Features Implemented

### 1. Scenario Generation
- Randomized initial parameters
- Three unique narrative templates
- Hidden variables (market confidence, political pressure, supplier trust)

### 2. Decision Engine
- 6 distinct actions with real tradeoffs
- Immediate, secondary, and cascade effects
- Cost-benefit modeling

### 3. Event System
- 5 stochastic events (port closures, supplier failures, etc.)
- Probability-based triggering
- Cumulative impact modeling

### 4. State Management
- 5 core KPIs tracked per turn
- Real-time updates
- Turn-by-turn history

### 5. Feedback Engine
- Final score calculation (4-factor model)
- Decision timeline analysis
- Behavioral bias detection
- Optimal strategy comparison
- Counterfactual simulation
